## String
1. **基础准备**
    
    - 头文件：`#include <string>`（必须）
    - 命名空间：`using namespace std;`（新手直接用，避免写`std::`）
    - 定义：`string s;`（空字符串）/ `string s = "abc";`（直接赋值）
    
2. **核心操作（3 个最常用）**
    
    - 取长度：`s.size()`（比如`"abc".size()`返回 3）
    - 拼接：`s += "123"`（直接追加，无需手动管内存）
    - 访问字符：`s[0]`（和数组一样，比如`"abc"[0]`是 'a'）
    
3. **输入输出（2 种核心场景）**
    
    - 读整行（含空格）：`getline(cin, s)`（推荐，适配绝大多数场景）
    - 输出：`cout << s;`（直接输出，不用格式化符）
    
4. **核心优势（对比 C 字符数组）**
    
    - 不用手动指定长度，自动扩容，不会溢出
    - 比较直接用`==`/`!=`（比如`s1 == s2`），不用记`strcmp`
    

### 总结

1. 必备：头文件`<string>` + 命名空间`std`，定义直接`string s = "内容"`；
2. 常用：`size()`取长度、`+=`拼接、`getline`读整行、`cout`输出；
3. 核心优势：自动管理内存，操作比 C 字符数组简单 10 倍。

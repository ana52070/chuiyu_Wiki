# 结构体

## 一、C 语言中结构体的定义和使用

在 C 语言中，定义结构体后，声明变量时必须使用 `struct` 关键字：

```c
#include <stdio.h>
#include <string.h>

// 定义结构体
struct Student {
    char name[50];
    int age;
    double score;
};

int main() {
    // 声明变量时必须加 struct
    struct Student s1;

    // 逐个成员赋值
    strcpy(s1.name, "Alice");
    s1.age = 20;
    s1.score = 95.5;

    // 初始化列表
    struct Student s2 = {"Bob", 21, 87.0};

    // 输出
    printf("姓名: %s, 年龄: %d, 成绩: %.1f\n", s1.name, s1.age, s1.score);
    printf("姓名: %s, 年龄: %d, 成绩: %.1f\n", s2.name, s2.age, s2.score);

    // 结构体赋值（值复制）
    struct Student s3 = s1;

    return 0;
}
```

C 语言结构体的局限：
- 不能包含成员函数
- 没有访问权限控制（public/private）
- 必须使用 `struct` 关键字声明变量
- 不能直接使用 `=` 进行整体操作（除非赋值）

## 二、C++ 结构体的简化

C++ 中，结构体类型名可以直接使用，**无需 `struct` 关键字**：

```cpp
#include <iostream>
#include <string>
using namespace std;

// C++ 结构体定义
struct Student {
    string name;    // 可以直接用 string 类型
    int age;
    double score;
};

int main() {
    // 声明变量时无需 struct 关键字
    Student s1;

    s1.name = "Alice";
    s1.age = 20;
    s1.score = 95.5;

    // 初始化列表
    Student s2 = {"Bob", 21, 87.0};

    // C++11 统一初始化
    Student s3 {"Charlie", 22, 91.0};

    cout << "姓名: " << s1.name << ", 年龄: " << s1.age
         << ", 成绩: " << s1.score << endl;
    cout << "姓名: " << s2.name << ", 年龄: " << s2.age
         << ", 成绩: " << s2.score << endl;
    cout << "姓名: " << s3.name << ", 年龄: " << s3.age
         << ", 成绩: " << s3.score << endl;

    return 0;
}
```

**运行结果**：
```
姓名: Alice, 年龄: 20, 成绩: 95.5
姓名: Bob, 年龄: 21, 成绩: 87.0
姓名: Charlie, 年龄: 22, 成绩: 91.0
```

## 三、结构体可以定义成员函数

C++ 结构体最大的增强：**可以定义成员函数**。

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    // 数据成员
    string name;
    int age;
    double score;

    // 成员函数：打印信息
    void printInfo() {
        cout << "姓名: " << name << ", 年龄: " << age
             << ", 成绩: " << score << endl;
    }

    // 成员函数：判断是否及格
    bool isPassed() {
        return score >= 60.0;
    }

    // 成员函数：修改成绩
    void setScore(double s) {
        if (s >= 0 && s <= 100) {
            score = s;
        }
    }
};

int main() {
    Student s1 = {"Alice", 20, 95.5};

    // 调用成员函数
    s1.printInfo();
    cout << "是否及格: " << (s1.isPassed() ? "是" : "否") << endl;

    s1.setScore(58.0);
    s1.printInfo();
    cout << "修改后是否及格: " << (s1.isPassed() ? "是" : "否") << endl;

    return 0;
}
```

**运行结果**：
```
姓名: Alice, 年龄: 20, 成绩: 95.5
是否及格: 是
姓名: Alice, 年龄: 20, 成绩: 58
修改后是否及格: 否
```

## 四、struct 与 class 的区别

C++ 中 `struct` 和 `class` 几乎完全相同，唯一的区别是**默认访问权限**：

- `struct`：默认成员是 **public**（公有）
- `class`：默认成员是 **private**（私有）

```cpp
#include <iostream>
using namespace std;

// struct — 默认 public
struct Point {
    int x;                  // public（默认）
    int y;                  // public（默认）
};

// class — 默认 private
class Rectangle {
    int width;              // private（默认）
    int height;             // private（默认）
public:
    Rectangle(int w, int h) : width(w), height(h) {}
    int area() {
        return width * height;
    }
};

int main() {
    Point p {10, 20};       // 可以直接访问 x, y
    cout << "Point: (" << p.x << ", " << p.y << ")" << endl;

    Rectangle r(5, 3);
    cout << "矩形面积: " << r.area() << endl;
    // cout << r.width;     // 编译错误：width 是 private

    return 0;
}
```

**运行结果**：
```
Point: (10, 20)
矩形面积: 15
```

### struct vs class 选用建议

| 场景 | 建议使用 |
|------|---------|
| 只有数据，没有或很少成员函数 | `struct` |
| 简单的 POD（Plain Old Data）类型 | `struct` |
| 需要封装和数据隐藏 | `class` |
| 需要继承和多态 | `class` |
| 算法竞赛中的简单结构 | `struct` |

## 五、结构体的构造函数

C++ 结构体可以有构造函数，与类完全相同：

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Point {
    double x, y;

    // 默认构造函数
    Point() : x(0), y(0) {}

    // 带参构造函数
    Point(double x_val, double y_val) : x(x_val), y(y_val) {}

    // 拷贝构造函数
    Point(const Point& other) : x(other.x), y(other.y) {}

    double distance() {
        return sqrt(x * x + y * y);
    }
};

int main() {
    Point p1;                   // 默认构造： (0, 0)
    Point p2(3.0, 4.0);        // 带参构造： (3, 4)
    Point p3 = p2;              // 拷贝构造

    cout << "p1: (" << p1.x << ", " << p1.y << ")" << endl;
    cout << "p2: (" << p2.x << ", " << p2.y << "), 距离原点: "
         << p2.distance() << endl;

    return 0;
}
```

**运行结果**：
```
p1: (0, 0)
p2: (3, 4), 距离原点: 5
```

## 六、结构体作为函数参数

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

struct Student {
    string name;
    int age;
    double score;
};

// 传值（复制一份）
void printStudent(Student s) {
    cout << s.name << " " << s.age << " " << s.score << endl;
}

// 传引用（修改原对象）
void addBonus(Student& s, double bonus) {
    s.score += bonus;
    if (s.score > 100) s.score = 100;
}

// const 引用（不复制，也不修改）
void displayStudent(const Student& s) {
    cout << "姓名: " << s.name << ", 成绩: " << s.score << endl;
}

int main() {
    Student s {"Alice", 20, 85.0};

    addBonus(s, 10.0);       // 修改原对象
    displayStudent(s);       // const 引用，只读

    // 结构体数组
    vector<Student> students = {
        {"Alice", 20, 95.0},
        {"Bob", 21, 87.0},
        {"Charlie", 22, 91.0}
    };

    cout << "\n所有学生:\n";
    for (const auto& stu : students) {
        displayStudent(stu);
    }

    return 0;
}
```

**运行结果**：
```
姓名: Alice, 成绩: 95
所有学生:
姓名: Alice, 成绩: 95
姓名: Bob, 成绩: 87
姓名: Charlie, 成绩: 91
```

## 七、完整可运行代码：综合展示

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

// 定义一个表示矩形的结构体
struct Rectangle {
    double width;
    double height;

    // 构造函数
    Rectangle(double w, double h) : width(w), height(h) {}

    // 成员函数
    double area() const {
        return width * height;
    }

    double perimeter() const {
        return 2 * (width + height);
    }

    // 是否正方形
    bool isSquare() const {
        return width == height;
    }

    // 打印信息
    void print() const {
        cout << "矩形 " << width << "x" << height
             << "，面积: " << area()
             << "，周长: " << perimeter();
        if (isSquare()) {
            cout << " （正方形）";
        }
        cout << endl;
    }
};

// 定义一个表示学生的结构体（用于排序示例）
struct Student {
    string name;
    int age;
    double score;
};

int main() {
    cout << "=== 结构体综合演示 ===\n\n";

    // 1. 矩形演示
    cout << "1. 矩形结构体:\n";
    vector<Rectangle> rects = {
        Rectangle(3.0, 4.0),
        Rectangle(5.0, 5.0),
        Rectangle(2.5, 6.0)
    };

    double total_area = 0;
    for (const auto& r : rects) {
        r.print();
        total_area += r.area();
    }
    cout << "总面积: " << total_area << "\n\n";

    // 2. 学生排序
    cout << "2. 学生排序（按成绩降序）:\n";
    vector<Student> students = {
        {"Alice", 20, 95.0},
        {"Bob", 21, 87.0},
        {"Charlie", 19, 92.5},
        {"David", 22, 88.0}
    };

    // 使用 lambda 表达式排序
    sort(students.begin(), students.end(),
         [](const Student& a, const Student& b) {
             return a.score > b.score;  // 降序
         });

    for (const auto& s : students) {
        cout << s.name << " " << s.age << " 岁 成绩: " << s.score << endl;
    }

    // 3. 结构体嵌套
    cout << "\n3. 结构体嵌套:\n";
    struct Address {
        string city;
        string street;
        int number;
    };

    struct Person {
        string name;
        int age;
        Address addr;

        void print() const {
            cout << name << ", " << age << " 岁\n"
                 << "  地址: " << addr.city << " "
                 << addr.street << " " << addr.number << "号\n";
        }
    };

    Person p = {"Alice", 20, {"北京", "中关村大街", 10}};
    p.print();

    return 0;
}
```

**运行结果**：
```
=== 结构体综合演示 ===

1. 矩形结构体:
矩形 3x4，面积: 12，周长: 14
矩形 5x5，面积: 25，周长: 20 （正方形）
矩形 2.5x6，面积: 15，周长: 17
总面积: 52

2. 学生排序（按成绩降序）:
Alice 20 岁 成绩: 95
Charlie 19 岁 成绩: 92.5
David 22 岁 成绩: 88
Bob 21 岁 成绩: 87

3. 结构体嵌套:
Alice, 20 岁
  地址: 北京 中关村大街 10号
```

## 八、注意事项和常见坑

### 8.1 结构体的内存对齐

```cpp
#include <iostream>
using namespace std;

struct A {      // 成员大小为 1+8+4 = 13，实际占用更大
    char c;     // 1 字节
    double d;   // 8 字节（需要 8 字节对齐）
    int i;      // 4 字节
};

struct B {      // 按大小降序排列，减少填充
    double d;   // 8 字节
    int i;      // 4 字节
    char c;     // 1 字节
};

int main() {
    cout << "sizeof(A): " << sizeof(A) << endl;  // 通常 24
    cout << "sizeof(B): " << sizeof(B) << endl;  // 通常 16
    return 0;
}
```

**运行结果**：
```
sizeof(A): 24
sizeof(B): 16
```

为了减少内存浪费，建议将结构体成员按**从大到小**的顺序排列。

### 8.2 结构体中的指针需要深拷贝

```cpp
struct BadExample {
    char* data;  // 指针成员，默认拷贝只复制指针值（浅拷贝）
    // 析构时两个对象可能指向同一内存，导致重复释放
};
```

### 8.3 大结构体传参使用 const 引用

```cpp
// 低效：每次调用都复制整个结构体
void process(LargeStruct s);

// 高效：const 引用，不复制
void process(const LargeStruct& s);
```

### 8.4 初始化注意事项

```cpp
struct Point {
    int x, y;
    // 如果有构造函数，则不能使用聚合初始化
    Point(int a, int b) : x(a), y(b) {}
};

// Point p = {1, 2};       // 错误！有构造函数后不能使用聚合初始化
Point p(1, 2);              // 正确：调用构造函数
```

## 九、总结

| 特性 | C 语言 struct | C++ struct |
|------|--------------|------------|
| 声明变量 | 必须加 `struct` | 直接使用类型名 |
| 成员函数 | 不支持 | 支持 |
| 构造函数 | 不支持 | 支持 |
| 访问权限 | 无（全部公有） | 可控制（默认 public） |
| 与 class 区别 | 无 | 默认访问权限不同 |
| 继承 | 不支持 | 支持（通常不用） |

**核心要点**：
1. C++ 结构体声明变量无需 `struct` 关键字
2. C++ 结构体可以包含成员函数和构造函数
3. `struct` 默认 public，`class` 默认 private，其他完全相同
4. 算法竞赛中适合用 `struct` 定义简单数据结构

# 引用和传值

## 一、传值（值传递）

### 1.1 原理

传值时，函数**复制一份参数副本**，函数内对副本的修改不影响原始变量。

```cpp
#include <iostream>
using namespace std;

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
    cout << "函数内部: a = " << a << ", b = " << b << endl;
}

int main() {
    int x = 10, y = 20;

    cout << "交换前: x = " << x << ", y = " << y << endl;

    swap(x, y);  // 传递的是 x 和 y 的副本

    cout << "交换后: x = " << x << ", y = " << y << endl;

    return 0;
}
```

**运行结果**：
```
交换前: x = 10, y = 20
函数内部: a = 20, b = 10
交换后: x = 10, y = 20
```

函数内部的 `a` 和 `b` 是 `x` 和 `y` 的**副本**，交换它们不影响原变量。

### 1.2 内存示意图

```
main():
  x = 10 ──────┐
  y = 20 ──────┤
               │
swap():        │ (复制)
  a = 10 ←─────┘
  b = 20 ←──────
  交换后：
  a = 20, b = 10  (不影响 x, y)
```

## 二、传引用（引用传递）

### 2.1 原理

传引用时，函数直接操作**原始变量**，不复制。引用本质上是变量的**别名**。

```cpp
#include <iostream>
using namespace std;

void swap(int &a, int &b) {  // & 表示引用
    int temp = a;
    a = b;
    b = temp;
    cout << "函数内部: a = " << a << ", b = " << b << endl;
}

int main() {
    int x = 10, y = 20;

    cout << "交换前: x = " << x << ", y = " << y << endl;

    swap(x, y);  // 传递的是引用

    cout << "交换后: x = " << x << ", y = " << y << endl;

    return 0;
}
```

**运行结果**：
```
交换前: x = 10, y = 20
函数内部: a = 20, b = 10
交换后: x = 20, y = 10
```

### 2.2 内存示意图

```
main():
  x = 10  <────引用──── swap() 中的 a 就是 x
  y = 20  <────引用──── swap() 中的 b 就是 y
  a 和 b 不占用额外内存，就是 x 和 y 本身
```

## 三、引用 vs 指针的对比

### 3.1 引用基础

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;

    // 引用：变量的别名
    int &ref = x;   // ref 是 x 的引用
    ref = 20;       // 修改 ref 就是修改 x

    cout << "x = " << x << endl;     // 20
    cout << "ref = " << ref << endl; // 20

    // 指针：保存变量的地址
    int *ptr = &x;   // ptr 指向 x
    *ptr = 30;       // 通过指针修改 x

    cout << "x = " << x << endl;     // 30
    cout << "*ptr = " << *ptr << endl; // 30

    return 0;
}
```

### 3.2 引用 vs 指针对比表

| 特性 | 引用 (`&`) | 指针 (`*`) |
|------|-----------|-----------|
| 语法 | `int &ref = x;` | `int *ptr = &x;` |
| 是否占用内存 | 不占用（别名） | 占用（通常是 4/8 字节） |
| 是否能为空 | **不能**（必须初始化） | 可以（nullptr） |
| 可重新绑定 | **不能**（初始化后不可改） | 可以（指向不同对象） |
| 解引用语法 | 直接使用 | 需要 `*` 运算符 |
| 取地址语法 | 自动 | 需要 `&` 运算符 |
| 安全性 | 更高 | 较低（空指针、野指针风险） |
| 适用场景 | 函数参数、返回值 | 动态内存、数组遍历、可选引用 |

### 3.3 对比代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 20;

    // ============ 引用 ============
    int &ref = x;    // 初始化时必须绑定
    ref = 100;       // 修改 x
    // &ref = y;     // 编译错误！引用不能重新绑定

    cout << "x = " << x << endl;       // 100
    cout << "ref = " << ref << endl;   // 100

    // ============ 指针 ============
    int *ptr = &x;   // 指向 x
    *ptr = 200;      // 通过指针修改 x

    cout << "x = " << x << endl;       // 200

    ptr = &y;        // 可以重新指向 y
    *ptr = 300;      // 修改 y

    cout << "y = " << y << endl;       // 300

    // 空指针检查
    int *null_ptr = nullptr;  // 合法
    // int &null_ref;         // 编译错误！引用必须初始化

    return 0;
}
```

**运行结果**：
```
x = 100
ref = 100
x = 200
y = 300
```

## 四、引用作为函数参数

```cpp
#include <iostream>
#include <string>
using namespace std;

// 1. 通过引用修改外部变量
void increment(int &n) {
    n++;  // 直接修改原变量
}

// 2. 通过引用实现多返回值
void minMax(int arr[], int size, int &minVal, int &maxVal) {
    minVal = maxVal = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] < minVal) minVal = arr[i];
        if (arr[i] > maxVal) maxVal = arr[i];
    }
}

// 3. const 引用：避免复制，同时保证不修改
void printMessage(const string &msg) {
    cout << msg << endl;
    // msg += "!";  // 编译错误！const 引用不能修改
}

int main() {
    // 测试 increment
    int a = 5;
    cout << "increment 前: " << a << endl;
    increment(a);
    cout << "increment 后: " << a << endl;

    // 测试 minMax
    int arr[] = {3, 1, 4, 1, 5, 9, 2};
    int minVal, maxVal;
    minMax(arr, 7, minVal, maxVal);
    cout << "最小值: " << minVal << ", 最大值: " << maxVal << endl;

    // 测试 const 引用
    string s = "Hello, World!";
    printMessage(s);

    return 0;
}
```

**运行结果**：
```
increment 前: 5
increment 后: 6
最小值: 1, 最大值: 9
Hello, World!
```

## 五、引用作为函数返回值

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 返回引用：可以修改数组中的元素
int& getElement(vector<int>& arr, int index) {
    return arr[index];  // 返回引用
}

// 返回 const 引用：只能读取，不能修改
const int& readElement(const vector<int>& arr, int index) {
    return arr[index];
}

int main() {
    vector<int> nums = {10, 20, 30, 40, 50};

    // 通过引用返回值修改元素
    getElement(nums, 2) = 999;  // 修改第3个元素
    cout << "修改后 nums[2] = " << nums[2] << endl;  // 999

    // 通过 const 引用读取（不能修改）
    const int& val = readElement(nums, 0);
    cout << "nums[0] = " << val << endl;
    // val = 100;  // 编译错误

    // 链式调用
    getElement(nums, 0) = getElement(nums, 1) = 100;
    cout << "链式赋值后: ";
    for (int n : nums) cout << n << " ";
    cout << endl;

    return 0;
}
```

**运行结果**：
```
修改后 nums[2] = 999
nums[0] = 100
链式赋值后: 100 100 999 40 50 
```

## 六、const 引用（const T&）的妙用

### 6.1 避免大对象复制

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

struct BigObject {
    int data[1000];  // 约 4KB
};

// 传值：复制整个对象，开销大
void processByValue(BigObject obj) {
    // ...
}

// 传 const 引用：不复制，只读访问
void processByConstRef(const BigObject& obj) {
    // ...
    // obj.data[0] = 42;  // 编译错误
}

int main() {
    BigObject obj;
    processByConstRef(obj);  // 高效：不复制
    cout << "const 引用传参完成" << endl;
    return 0;
}
```

### 6.2 const 引用可以延长临时对象的生命周期

```cpp
#include <iostream>
#include <string>
using namespace std;

string makeGreeting() {
    return "Hello, World!";
}

int main() {
    // const 引用可以绑定到临时对象，并延长其生命周期
    const string &ref = makeGreeting();
    cout << ref << endl;  // 正常访问

    // 普通引用不能绑定到临时对象
    // string &ref2 = makeGreeting();  // 编译错误！

    // 实用场景：函数参数接收字符串字面量
    void process(const string &s);
    process("temporary string");  // const string& 可以接收字面量

    return 0;
}
```

### 6.3 const 引用的常见陷阱

```cpp
int main() {
    // 正确：不同类型的临时对象也可以绑定
    const double &d = 42;      // 隐式创建临时 double(42)，然后绑定
    cout << d << endl;         // 42.0

    // 危险：返回局部变量的引用
    int& getLocal() {
        int x = 10;
        return x;  // x 是局部变量，函数结束时销毁
    }
    // int &r = getLocal();  // 悬空引用！未定义行为
    // cout << r;           // 危险！

    return 0;
}
```

## 七、完整可运行代码：传值、传引用、传指针对比

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

// ============ 三种参数传递方式对比 ============

// 1. 传值：复制一份副本
void tripleByValue(int x) {
    x *= 3;
    cout << "  函数内: x = " << x << endl;
}

// 2. 传引用：操作原变量
void tripleByReference(int &x) {
    x *= 3;
    cout << "  函数内: x = " << x << endl;
}

// 3. 传指针：通过地址修改
void tripleByPointer(int *x) {
    if (x) {          // 需要检查空指针
        *x *= 3;
        cout << "  函数内: *x = " << *x << endl;
    }
}

// ============ 对比大对象传参 ============
struct LargeData {
    int data[10000];  // 约 40KB
    long long sum() {
        long long s = 0;
        for (int i = 0; i < 10000; i++) s += data[i];
        return s;
    }
};

// 传值：会复制整个大对象（低效）
long long sumByValue(LargeData d) {
    return d.sum();
}

// 传 const 引用：不复制（高效）
long long sumByRef(const LargeData &d) {
    return d.sum();
}

int main() {
    cout << "=== 三种传递方式对比 ===\n\n";

    // ---- 测试 1：传值 ----
    cout << "1. 传值:\n";
    int a = 5;
    cout << "  调用前: a = " << a << endl;
    tripleByValue(a);
    cout << "  调用后: a = " << a << " (不变)\n\n";

    // ---- 测试 2：传引用 ----
    cout << "2. 传引用:\n";
    int b = 5;
    cout << "  调用前: b = " << b << endl;
    tripleByReference(b);
    cout << "  调用后: b = " << b << " (改变)\n\n";

    // ---- 测试 3：传指针 ----
    cout << "3. 传指针:\n";
    int c = 5;
    cout << "  调用前: c = " << c << endl;
    tripleByPointer(&c);
    cout << "  调用后: c = " << c << " (改变)\n\n";

    // ---- 测试 4：多返回值 ----
    cout << "4. 引用实现多返回值:\n";
    vector<int> arr = {5, 2, 8, 1, 9, 3, 7};
    int minimum, maximum;
    // 通过引用参数返回多个值
    auto findMinMax = [&](const vector<int>& v, int &mn, int &mx) {
        mn = mx = v[0];
        for (size_t i = 1; i < v.size(); i++) {
            if (v[i] < mn) mn = v[i];
            if (v[i] > mx) mx = v[i];
        }
    };
    findMinMax(arr, minimum, maximum);
    cout << "  最小值: " << minimum << ", 最大值: " << maximum << "\n\n";

    // ---- 测试 5：引用范围 for 循环 ----
    cout << "5. 引用在范围 for 中的应用:\n";
    cout << "  原数组: ";
    for (int x : arr) cout << x << " ";
    cout << "\n";

    // 使用引用修改数组元素
    for (int &x : arr) {
        x *= 2;  // 所有元素翻倍
    }
    cout << "  翻倍后: ";
    for (int x : arr) cout << x << " ";
    cout << "\n\n";

    // ---- 测试 6：大对象传参（概念验证） ----
    cout << "6. 大对象传参:\n";
    LargeData ld;
    for (int i = 0; i < 10000; i++) ld.data[i] = i;

    // 下面两行只是概念演示，实际运行时间可忽略
    long long s1 = sumByValue(ld);    // 复制了 40KB
    long long s2 = sumByRef(ld);      // 没有复制
    cout << "  结果相同: " << s1 << ", " << s2 << "\n";
    cout << "  (sumByRef 效率更高, 不复制大对象)\n";

    return 0;
}
```

**运行结果**：
```
=== 三种传递方式对比 ===

1. 传值:
  调用前: a = 5
  函数内: x = 15
  调用后: a = 5 (不变)

2. 传引用:
  调用前: b = 5
  函数内: x = 15
  调用后: b = 15 (改变)

3. 传指针:
  调用前: c = 5
  函数内: *x = 15
  调用后: c = 15 (改变)

4. 引用实现多返回值:
  最小值: 1, 最大值: 9

5. 引用在范围 for 中的应用:
  原数组: 5 2 8 1 9 3 7 
  翻倍后: 10 4 16 2 18 6 14 

6. 大对象传参:
  结果相同: 49995000, 49995000
  (sumByRef 效率更高, 不复制大对象)
```

## 八、注意事项和常见坑

### 8.1 不要返回局部变量的引用

```cpp
int& badFunction() {
    int local = 42;
    return local;   // 致命错误！local 在函数结束时销毁
}                   // 返回的引用变成了"悬空引用"

// 正确做法：返回全局/静态变量，或通过参数返回
int& goodFunction(int &x) {
    return x;       // 安全，x 在函数外仍然存在
}
```

### 8.2 引用必须初始化，不能为空

```cpp
int &ref;         // 编译错误！引用必须初始化
int &ref = x;     // 正确

int *ptr;         // 合法，但危险（野指针）
int *ptr2 = nullptr;  // 合法，空指针
```

### 8.3 引用一旦绑定，不能改变

```cpp
int a = 10, b = 20;
int &ref = a;
ref = b;        // 这不是重新绑定！这是将 a 赋值为 b 的值
// 等价于 a = b; 现在 a = 20，ref 仍然引用 a
```

### 8.4 警惕引用带来的副作用

```cpp
int x = 10;
int &r = x;
// 代码很长...
// 不小心使用了 r
r = 100;  // 不经意间修改了 x
```

### 8.5 const 引用可以延长临时对象生命周期（但谨慎使用）

```cpp
const int &r = 42;      // 合法，42 的临时对象会存活到 r 离开作用域
// int &r2 = 42;        // 编译错误！普通引用不能绑定到临时对象
```

## 九、总结

| 特性 | 传值 | 传引用 | 传指针 |
|------|------|--------|--------|
| 是否复制 | 是 | 否 | 否（复制地址） |
| 能否修改原变量 | 否 | 能 | 能 |
| 是否能为空 | N/A | 不能 | 能（nullptr） |
| 语法复杂度 | 低 | 低 | 较高 |
| 安全性 | 安全 | 安全 | 需检查空指针 |
| 大对象开销 | 大 | 小 | 小 |
| 是否需要取址 | 不需要 | 不需要 | 需要 `&` |

**核心建议**：
1. **需要修改参数**：使用引用（`int &x`）
2. **需要避免复制**：使用 const 引用（`const T &x`）
3. **需要可选值（可能为空）**：使用指针
4. **返回多个值**：使用引用参数
5. **范围 for 循环中修改元素**：使用引用（`for (auto &x : arr)`）
6. **范围 for 循环中只读访问大对象**：使用 const 引用（`for (const auto &x : arr)`）

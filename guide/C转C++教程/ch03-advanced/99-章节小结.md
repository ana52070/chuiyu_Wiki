# 第三章 进阶知识 — 章节小结

## 01 bitset（位运算）

`bitset` 是一个模板化的固定大小二进制位容器，所有位操作在编译期确定的位宽上进行，效率极高。核心要点：`b[0]` 是最低位（LSB），与直接输出的字符串顺序（左高右低）相反；常用成员函数包括 `any()`、`none()`、`all()`、`count()`、`set()`、`reset()`、`flip()`、`test()`，以及 `to_ulong()`/`to_string()` 等类型转换函数；`bitset` 还支持 `&`、`|`、`^`、`~`、`<<`、`>>`、`==`、`!=` 等运算符。

## 02 sort 函数

`std::sort` 基于内省排序（IntroSort）实现，是快速排序、堆排序和插入排序的混合体，平均和最坏时间复杂度均为 O(n log n)。它要求随机访问迭代器，适用于数组、`vector`、`deque` 等容器，但不支持 `list`（必须用 `list::sort()`）。默认按 `operator<` 升序排列，传入自定义比较函数（或 `greater<T>()`）可实现降序等其他排序规则。

## 03 自定义 cmp 函数

自定义比较函数的返回值表示第一个参数是否应排在第二个参数之前，返回 `true` 在前、`false` 在后。比较函数必须满足**严格弱排序**（Strict Weak Ordering）：只能用 `<` 或 `>`，**不能**用 `<=` 或 `>=`。cmp 有三种常见写法：普通函数、函数对象（仿函数）和 Lambda 表达式（现代 C++ 推荐）。对结构体排序时，可通过多字段比较（`if` 分支或 `std::tie`）实现复杂的排序逻辑。

## 04 cctype 头文件

`<cctype>` 是 C 语言 `<ctype.h>` 的 C++ 版本，提供了 12 个字符判断函数（`isalpha`、`islower`、`isupper`、`isdigit`、`isalnum`、`isspace`、`isblank`、`ispunct`、`isprint`、`isgraph`、`iscntrl`、`isxdigit`）和 2 个字符转换函数（`tolower`、`toupper`）。这些函数接受 `int` 参数并返回 `int`，使用时注意将 `char` 转换为 `unsigned char` 以避免符号扩展导致的未定义行为。

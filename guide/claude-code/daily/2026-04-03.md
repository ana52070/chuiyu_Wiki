## 今晚主题

Claude Code 和 Cursor 的核心区别。

## 今晚发出的内容

今晚先记一个最容易混的点：**Claude Code 不只是“终端版 Cursor”**。更顺手的理解是——Cursor 像坐在你旁边一起改代码的搭子，你盯着屏幕边聊边改；Claude Code 更像一个进了你项目环境、能自己去读文件、跑命令、改代码、测结果的终端代理。

比如你说“把这个登录模块重构一下，顺手把测试补齐”。用 Cursor 时，你通常还是会盯着 IDE 看它一段段改；但用 Claude Code，它更适合自己在终端里把整条链路跑下来。**所以两者差别不只是界面，而是你和 AI 的协作关系变了。**

今晚记住一句话：**Cursor 更像副驾驶，Claude Code 更像能自己下车干活的人。**
